# プライバシー

## データソース
- ユーザー環境内のローカル VRChat ログファイル

## データ処理
- ログ行をローカルルール（`contains` / `regex`）で評価
- マッチしたイベントをローカル JSONL イベントログへ出力
- 任意機能: マッチ内容を設定済み Discord Webhook へ送信
- 任意機能: マッチ内容をユーザー定義 Hook コマンドへ標準入力 / 引数で受け渡し

## 収集しないもの
- VRChat 公式 API 経由のデータ
- アカウントのログイン資格情報
- 本MVPにおける外部テレメトリ送信

## ユーザーによる制御
- Discord 通知 / Hook は無効化可能
- `--dry-run` で外部通知送信を抑止可能
- 設定ファイルはローカル保存・ユーザー編集可能

## 注意事項
VRChat ログには、利用状況や導入プラグインによって識別子や機微情報が含まれる可能性があります。  
マッチ条件と外部送信先は必ず確認してください。
